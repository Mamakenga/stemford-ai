## 1) Product ladder

The project is split into three non-conflicting product levels:
1. `Coder Factory`
2. `Business Control Plane`
3. `Reusable Service`

Rule:
1. First we build the coder factory.
2. Then the coder factory builds the Stemford business control plane.
3. Then we generalize the system into a reusable service for other businesses.
