# Ops 5-Line Regulation

1) Перед любой важной задачей в боте запусти: /техосмотр
2) Норма: одновременно есть Summary: PASS=12 FAIL=0 SKIP=0 и Summary: PASS=8 FAIL=0 SKIP=0
3) Если есть FAIL/PREFLIGHT_FAILED/BLOCKED_* — реальные задачи не запускать
4) После правок в CI/CD-скриптах снова запустить /техосмотр
5) Если техосмотр зеленый — можно запускать рабочий цикл
